# Bibliothèque Numérique Microservices

## Description

Projet DevOps basé sur une architecture microservices pour la gestion d'une bibliothèque numérique.

## Microservices

### Backend Livres

- Ajouter un livre
- Modifier un livre
- Supprimer un livre
- Consulter les livres

### Backend Utilisateurs

- Créer un utilisateur
- Modifier un utilisateur
- Supprimer un utilisateur
- Consulter les utilisateurs

### Backend Emprunts

- Emprunter un livre
- Retourner un livre
- Historique des emprunts
- Détection des retards

## Technologies

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Docker
- Docker Compose
- Git / GitHub
- Jenkins
